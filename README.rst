==========
SVD-Python
==========


Projekt na zajęcia "Język Python w uczeniu maszynowym" - algorytm SVD


Opis
====

Projekt będzie polegał na implementacji algorytmu SVD (Singular Value Decomposition) czyli Rozkładu macierzy według wartości osobliwych.
